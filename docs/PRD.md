## 2. 产品目标

- 通过提问、反馈和复盘形成可持续的学习闭环。

- 产品经理求职者

## 4. 核心使用场景

1. 产品案例分析训练
2. PRD 撰写与评审
3. 产品面试模拟
4. 数据分析与决策训练
